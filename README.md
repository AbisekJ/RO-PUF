# Ring Oscillator PUF for Secure RISC-V

## Overview
Designed and implemented an 8-bit Ring Oscillator Physically Unclonable Function for hardware root-of-trust applications.

## Features
- Device-unique fingerprint generation
- Challenge-response generation
- Integrated with RV32I processor
- FPGA validation across multiple boards

## Methodology
- RO-based entropy generation
- Response stability evaluation
- Inter-device uniqueness analysis
- Intra-device reliability characterization

## Tools
- Verilog
- Xilinx Vivado
- Arty A7

## Results
Validated stable and unique PUF responses across 5 FPGA boards.

## Applications
- Secure boot
- Key generation
- Hardware authentication
